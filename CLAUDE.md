## 此專案是什麼 / What this project is

**LCP（Lobster Communication Protocol）v3.4** — 本地 7B 模型極致壓縮通訊協議。
2368 行 Python，105 個測試，6 指令集（CA/MB/SK/RM/RP/EA）。

## 常用指令 / Quick Commands

```bash
python lcp.py test           # 跑全部 105 個測試
python lcp.py chat "..."     # 自然語言模式
python lcp.py hybrid "..."   # Hybrid 模式（對外自然語言）
python lcp.py mem export     # 匯出核心記憶
```
